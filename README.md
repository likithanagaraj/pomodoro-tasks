## Getting Started

Follow these steps to set up the project on your local machine.

### 1. Clone the repository
```bash
git clone https://github.com/likithanagaraj/pomodoro-tasks.git
cd pomodoro-tasks
```

### 2. Intall Dependencies

##### Using npm
``` bash
npm install
```

##### OR using bun
``` bash
bun install
```


### 3. Run the project
``` bash
npm expo start
```
or
``` bash
bun expo start
```
